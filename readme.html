Project Installation:

  1. find out the database named with mdot.sql in root. import this into db
  2. Admin Username/password: admin/admin@1234
  3. Guest Username/Password: guest/guest@1234
  4. Once you logged as admin , please go to the Guest Post Submission Plugin Setting . use mentioned shortcode on any page to show posts/ form

Plugin Uses:

  1. You can directly use guest-post-submission plugin in any wordpess theme you need only few things already.
    1. A Guest author user
    2. Custom post type already exist in theme.
    3. follow the plugin instructions mention in plugin settings
